VINYL CUTTING

In this assignment, I will be printing some text and a graphic on my hoodie using the Vinyl cutter and the heat press.

8.1 Desinging

Using the "Sure Cuts A Lot" software, I was able to write a text and upload an image and the sofware turned it into objects to cut out. Initially I wrote a phrase longer than 15 characters. The problem was that this Software doesn't allow new lines in one text object. Hence I had to divide my text in to 3 text objects. This also allowed me to select them seperately.

However, the text is not properly aligned and I want it to be aligned from their center. For this I selected all 3 Text object and I used the "Allign to the Middle Horizontally" tool.

After the Aligning to the middle, it looks like this:

After centering them and allinging the texts as I want, now to prepare for cutting I have to place my objects to the top left corner of the software. This is because the top left corner is the home point of the software. So, it will make it easier for me to visualize where the object will go and I will waste as less material as possible. However, before doing that I need to combine all the 3 texts to one single object. This will allow me to change the location to top left without actually messing the allign. So I select all 3 objects and group them using the "group" tool.

Now that I groupped them, I can use the "Allign to the top side" and "Allign to the left side" tools to put my object to the top left corner of the software.

After alligning the object to the top left, I had to put margins between the corners of the software and the object. This secures that my object will be cut inside the boundaries of my vinyl. Before and After pics:

However, if I were to cut my text like this on vinyl cutter, when I put my vinyl on my hoodie and then heat press it, the text will be inverse. So I have to use "Flip Horizontal" tool to flip my text so that when I press it on my hoodie it will be in correct orientation.

Now that I have my text done, I need an image for my hoodie. To match with the text, I found a monochrome colored fish picture and uploaded it to the software

To upload the image to the software, I initially downloaded the image and using the "Trace Image" tool I was able to upload the image. Image tracing tool takes in an image and throws out the boundaries of the image showing where will the vinyl cutter will cut.

After I click "OK" it put the fish image on to a random place instead of putting it under my text while cutting I put it right next to it because the viynl I am using has 25 inches in length and in total the text and the fish pic side by side has 16 inches in length.

Now that every thing is ready, I can use the "Cut" tool to print (cut) my object in Vinyl Cutter. When I click it gives me a prieview of what the cut will be like.
