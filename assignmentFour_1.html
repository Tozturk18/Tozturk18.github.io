4.2 Uploading to CAM

First I need to import our STL file to the Prusas CAM. For this I just dragged the STL file and dropped it on the prusas app. Then I checked the dimesions of my design and if there is any hole in my design. If there is holes in my design that can prevent the "Vase" mode in Prusas to work properly.

After checking that everything is correct, I open the printer settings and check the Spiral Vase button in the vertical shells section. This will indicate the Prusas that I am going to use the Vase mode.

If I go back to see my object it is still the same. However once I click the butto "slice" on the bottom left corner of Prusas, it shows how the printer will print it:

Then I click "g-slice" and this will make a new file that the 3D printer can read to print my object. After I print out my Flower Pot this is how it looks like:
