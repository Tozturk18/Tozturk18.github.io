LASER CUTTING

In this assignment, I will be creating a storage box with elastic parts that allows me to open and close the lid without classical hinges.

2.1 Preparing for Laser Cutting

For creating this box, instead of designing it myself, I used "boxes.py" webpage that has set elements that I can play with to create a box design with "living" hinges.

I used the initial settings given by the website and I clicked "generate" for the website to generate an SVG file that has the exploded drawing of my box in 2D surface. This enables me to cut a 2D piece and combine those 2D pieces to make a 3D box. Moreover, the website generated it as a SVG file and this allowed me to use "Adobe Illustrator" for editing the file. Hence I edited so that it will have my name on the front face and so that all the pieces are aligned to the top left corner.

Then I divided that file to two different files. The first one has all the sides except the front part (the side with thee name on) and the bottom part (the perfect square part). The second file only has the front and the bottomp parts. I seperated this to two different files because I wanted to laser cut first two parts of the box to see if the hinges are fitting together or not.

After preparing my file, I was still unsure about which material to use and which settings to use for that material. Since our laser cutter is really old the output power isn't the same as a brand new one. Because of this I decided on cutting squares out of 4 different materials and engraving the settings that worked for them.
For this I used Acrylic, 1/4th maple wood, 1/8th ply wood, Cardboard:

Now that I know the settings for these 4 materials, I can chose one to use for my living hinge box. I chose 1/8th ply wood due to its strength to thickness ratio.
