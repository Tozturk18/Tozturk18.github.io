LASER CUTTING

In this assignment, I will be creating a storage box with elastic parts that allows me to open and close the lid without classical hinges.

4.1 Designing for Laser Cutting

For creating this box, instead of designing it myself I used "boxes.py" webpage that has set elements that I can play with to create a box design with "living" hinges.

I used the initial settings given by the website and I clicked "generate" for the website to generate an SVG file that has the exploded drawing of my box in 2D surface. This enables me to cut a 2D piece and combine those 2D pieces to make a 3D box. Moreover, using a 2D drawing enables me to use "Adobe Illustrator" for editing the file. Hence I edited so that it will have my name.
